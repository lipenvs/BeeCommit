Your privacy matters. Here's exactly what BeeCommit does and doesn't do.

Effective Date: February 10, 2026

1. Overview

BeeCommit is a Chrome extension that helps you sync your accepted Beecrowd solutions to a GitHub repository. This privacy policy explains how data is handled by the extension.

2. Data We Collect

BeeCommit collects and stores the following data locally in your browser:

  • GitHub Personal Access Token (PAT) ‚Äî used to authenticate with the GitHub API.
  • Repository name and owner ‚Äî used to determine where to commit your solutions.
  • Extension preferences ‚Äî such as auto-sync toggle state.

All data is stored using chrome.storage.sync, which is encrypted by the browser and accessible only to the extension itself.

3. Data We Do NOT Collect

  • We do not collect any personal information (name, email, etc.).
  • We do not track your browsing history or activity.
  • We do not use cookies, analytics, or any third-party tracking services.
  • We do not send any data to our own servers ‚Äî the extension has no backend.

4. Third-Party Services

BeeCommit communicates directly with the following services:

  • GitHub API (api.github.com) ‚Äî to validate your token, list repositories, and create/update files. Communication is encrypted via HTTPS.
  • Beecrowd (judge.beecrowd.com) ‚Äî the extension reads page content (problem ID, language, source code) from Beecrowd pages you visit. It does not send any data to Beecrowd.

5. Data Sharing

BeeCommit does not share, sell, or transfer your data to any third party. Your GitHub token is used exclusively for GitHub API calls initiated by you.

6. Data Storage and Security

  • Your GitHub token is stored in the browser's encrypted storage (chrome.storage.sync).
  • The token is never exposed to web pages, injected scripts, or external servers.
  • We recommend using a Fine-grained Personal Access Token scoped to a single repository with minimal permissions (Contents: Read and write).

7. Your Control

You have full control over your data:

  • You can view, update, or delete your stored token at any time via the extension's popup.
  • Uninstalling the extension removes all locally stored data.
  • You can revoke your GitHub token at any time from GitHub Settings.

8. Changes to This Policy

If this privacy policy is updated, the changes will be reflected on this page with an updated effective date. Significant changes will be communicated via the extension's release notes.

9. Contact

If you have any questions or concerns about this privacy policy, please open an issue on the GitHub repository.
